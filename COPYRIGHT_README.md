# TrainKit Copyright Protection Guide

## Overview

TrainKit is proprietary software owned by **TrainKit Ltd**. All source code, documentation, designs, databases, and related materials are protected by:

- **Copyright Law** (UK Copyright, Designs and Patents Act 1988)
- **Trade Secret Law**
- **Contract Law** (License Agreements)
- **International Treaties** (Berne Convention, WIPO Copyright Treaty)

### What is Protected:
✅ All source code (TypeScript, JavaScript, React, Next.js)
✅ Database schemas and structures (Prisma models)
✅ UI/UX designs and components
✅ Business logic and algorithms
✅ API endpoints and data structures

### What Users CANNOT Do:
❌ Copy the source code
❌ Reverse engineer the software
❌ Create competing products based on TrainKit
❌ Resell or redistribute the software
❌ Remove copyright notices
❌ Share access credentials
❌ Extract and reuse components

## Legal Enforcement

### Civil Remedies:
- Injunctive relief (court order to stop infringement)
- Monetary damages for losses
- Account of profits (infringer must pay profits earned)
- Legal costs recovery

### Criminal Penalties (UK):
- Up to 10 years imprisonment for serious copyright infringement
- Unlimited fines
- Criminal record

## How to Strengthen Protection

### 1. Register Your Copyright (Optional but Recommended)
While copyright is automatic in the UK, registration provides additional evidence:
- **UK Intellectual Property Office**: https://www.gov.uk/topic/intellectual-property/copyright
- Cost: £0 (automatic) or £200-500 for formal registration service

### 2. Register Your Trademark
Protect the "TrainKit" name and logo:
- **UK IPO Trademark Registration**: https://www.gov.uk/how-to-register-a-trade-mark
- Cost: £170 for one class, £50 for each additional class
- Provides exclusive rights to the name

### 3. Use Non-Disclosure Agreements (NDAs)
When showing the software to:
- Potential investors
- Business partners
- Contractors or developers
- Always use an NDA first

### 4. Employee/Contractor IP Assignment
Ensure all work created by employees or contractors is assigned to TrainKit Ltd:
- Include IP assignment clauses in employment contracts
- Use contractor agreements with IP ownership clauses

### 5. Monitor for Infringement
- Set up Google Alerts for "TrainKit" and similar names
- Monitor GitHub and code repositories
- Check competitor websites for copied features

## If Someone Infringes Your Copyright

### Step 1: Document the Infringement
- Take screenshots with timestamps
- Archive web pages
- Save source code comparisons
- Gather evidence of copying

### Step 2: Send Cease and Desist Letter
- Formal letter demanding they stop
- Usually drafted by solicitor
- Cost: £500-1,500

### Step 3: Legal Action (if necessary)
- Consult intellectual property solicitor
- Consider:
  - Injunction (emergency court order)
  - Damages claim
  - Criminal prosecution (serious cases)

### Recommended Solicitors (UK):
- **Slaughter and May** (London) - Major tech IP
- **Bristows** (London) - Specialist IP firm
- **Mathys & Squire** - Software and tech IP
- **Local IP solicitors** - Often more affordable for SMEs
